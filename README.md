This repository provides a matlab script to help you compile cuda source files.

# Usage

```matlab
 compilecuda('file1.cpp file2.cu file3.cu')
```

# Example

We provide an example in the path of `test`

```matlab
 cd test  
 addpath ..  
 compilecuda('test_mex.cpp test_gpu.cu')
```
